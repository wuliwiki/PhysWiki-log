http 包的结构
\begin{lstlisting}[language=none]
<method> <URI> HTTP/<version>
<Header1>: <value1>
<Header2>: <value2>
...
<HeaderN>: <valueN>
(空行)
<body>
\end{lstlisting}

\begin{itemize}
\item \verb|method| 有 \verb|GET, POST, PUT, DELETE, HEAD, OPTIONS, PATCH, CONNECT, TRACE|
\end{itemize}
