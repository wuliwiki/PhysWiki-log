\subsection{20分}

\[

\begin{cases}
0, & 0 \leq x \leq L \\
\infty, & \text{其他}
\end{cases}~
\]
中，有一质量为 $m$ 的粒子处于基态。在某一时刻，势阱突然变为 ：
\[
V_2(x) = 
\begin{cases}
0, & 0 \leq x \leq 2L \\
\infty, & \text{其他}
\end{cases}~
\]
(1) 分别求在 $V_1(x)$ 和 $V_2(x)$ 中粒子的基态能量和归一化波函数。\\
(2) 假设在势阱改变的瞬间，粒子的波函数并未发生变化，求粒子在 $V_2(x)$ 中仍处于基态的几率。

\subsection{20分}
用记一维简谐振子 $H_0 = \frac{1}{2}(p^2 + x^2)$ 的能量本征态，这里 $n=0,1,2,\dots$。
\begin{enumerate}
    \item 定义升降算符 $a^+ = \frac{x - ip}{\sqrt{2\hbar}}$，$a^- = \frac{x + ip}{\sqrt{2\hbar}}$，已知 $a^+ |n\rangle = \sqrt{n+1}|n+1\rangle$，$a^- |n\rangle = \sqrt{n} |n-1\rangle$，试将$x$用$a^+$和$a^-$表示，求出 $x^3|0\rangle$。
    
    \item 如果体系受到微扰 $H' = \lambda x^3$，其中 $\lambda$ 是小量，求基态能级的一阶修正和二阶修正。
\end{enumerate}
