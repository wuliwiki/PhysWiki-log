\subsection{光}
\begin{itemize}
\item 光是一种电磁波， 即电场和磁场的涟漪。
\item 光的传播不需要介质， 可以在真空中传播。
\item 真空光速（定义）： $299792458\Si{m/s}$。 约三十万千米每秒， 或三亿米每秒， 可以每秒钟绕地球七圈半。
\item 光的速度和参考系无关。
\item 光的红移和蓝移。
\end{itemize}
