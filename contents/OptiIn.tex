\subsection{波动光学}
\begin{itemize}
\item 光谱，阳光，三棱镜，彩虹
\item 光是电磁波， 就是电场和磁场产生的波
\item 可见光的范围是 400nm 到 700nm， nm 是 $1\e{-9}$， 绿光，550nm
\item 波长越长， 颜色越红， 波长越短颜色越紫
\item 电磁波的速度都是光速
\item 光源靠近： 蓝移。 远离： 红移。
\end{itemize}
