GNUGNU Scientific Library (GSL) 是一个 GNU 开源项目， 由 C 语言写成．

\subsection{安装}
在 Debian 系统上可执行 \verb|sudo apt install libgsl-dev| 安装．

第二种方法是自己下载源码编译．

\subsection{使用}
在编译时加上 linker 选项 \verb|-lgsl| 即可．
