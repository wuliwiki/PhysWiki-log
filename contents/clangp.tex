\begin{itemize}
\item clang 是一个比较现代的编译器， 一般的编程刷题网站都用 clang。
\item 支持四精度
\item 支持 address sanitizer
\item Clang 从设计的时候就是一个 API 也就是其他程序可以用它来做 code analysis 等。
\end{itemize}
